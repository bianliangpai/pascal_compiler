## Dependencies

### cmake
```bash
  sudo apt-get install cmake
```

### gtest
```bash
  cd /path/to/googletest <br/>
  mkdir build            <br/>
  cd build               <br/>
  cmake ..               <br/>
  make                   <br/>
  sudo make install      <br/>
```
### pthread
```bash
  sudo apt-get install libpthread-stubs0-dev
```
